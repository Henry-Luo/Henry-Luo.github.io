用 Python 来守护你的程序

之前项目里有个程序，含有 HTTP 上传的相关逻辑；长时间跑，偶尔会出现连接超时的异常，然后再也不能进行新的上传。经过多方面的排查，我最终定位到是该程序里使用的第三方 sdk 的问题，该 sdk 中在某种特定场景下没有正确释放连接，导致一段时间后，获取不到任何可用连接，于是有了连接超时的异常。
