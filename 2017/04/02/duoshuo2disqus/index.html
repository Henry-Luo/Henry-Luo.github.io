再见多说｜Hello Disqus

“多说” 是国内使用率很高的评论组件，然而前段时间，该评论组件变得不太稳定，且涌入大量垃圾广告评论，最重要的，它似乎没有盈利。于是这个结局也就不出意料了，多说在3月21号宣布即将关闭：重要通知: 多说即将关闭 。

这下毫不犹豫决定迁移到 Disqus 了。之前选择多说，一是因为本地化做得好，二是局域网内访问良好。可现在才发现，能长久提供稳定的服务才是最重要的，毕竟数据迁移的成本相当大。只是现在你可能需要一把梯子来访问 Disqus 了。

Disqus 接入很简单，和多说差不多，把 JS 代码插入到需要展示评论框的页面就行了。只是注意 url 和 identifier 的唯一性。

本以为迁移到 Disqus，原来的那些多说评论就丢了，因为二者不能直接互相导入导出。搜索一番，发现已有解决方案。思路是解析多说评论数据，再生成可导入到 Disqus 的 WordPress 评论结构 XML。这里分享一个方法：多说评论迁移至Disqus ，里面有使用方法介绍，感谢这位博主分享的方法。忽然想到，自己写一个程序解析的话，估计又得上 Java，这么一想还是脚本语言方便啊，看来得学个 PHP 或者 Python 了。

再见多说｜Hello Disqus .
